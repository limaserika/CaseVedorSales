# RPA Sales Report Automation

This project was developed as part of an RPA technical case.

## Overview
The automation extracts data from a PDF file, validates business rules, enriches data using external APIs, and generates personalized sales reports per vendor.

## Features
- PDF data extraction
- Business rules validation
- Currency conversion (USD to BRL via API)
- Address enrichment (Correios)
- Error reporting
- Automated Excel report generation
- PDF generation with password protection
- Automated email delivery with attachments

## Technologies
- UiPath
- C#
- REST API
- Excel Automation

## Output
- Sales reports per vendor (PDF with password)
- Error report (Excel)
- Automated email with attachments
